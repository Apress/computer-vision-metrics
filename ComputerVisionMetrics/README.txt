Additional README.txt files are provided in each directory shown below.
Read all README files, and follow instructions to build and test the code:
-------------------------------------------------------------------
Chapter3_SDM_ExtendedMetrics/sdmx 
- contains source code for extended SDM metrics (from book Chapter 3)

AppendixA_SyntheticAlphabetDetection/*
- /BinningHistograms - contains 2D histogram images showing detector summary counts
- /images - contains corner point and interst point alphabets
- /code/opencv_tests - contains source code for modifying opencv detectors tests to evaluate the corner and interest point alphabets
- /scripts - contains shell scripts for driving the tests, creating summary reports, and creating 2D histogram sets
